NAME

ookdimensions − reports the volume size given when initializing the file.

SYNOPSIS

#include <errno.h>
#include <ook.h>

void ookdimensions(const struct ookfile* of, uint64_t dims[3]);

DESCRIPTION

ookdimensions() reports the volume size which was input in the previous ookread(3) or ookcreate(3) call. The dims argument is filled with the volume size.

RETURN VALUE

ookdimensions() technically returns nothing. Instead, it fills the dims argument. On error, errno is set and the contents of dims are undefined.

ERRORS

EINVAL

The given of pointer does not point to a valid struct ookfile pointer.

SEE ALSO

ookread(3), ookcreate(3)
