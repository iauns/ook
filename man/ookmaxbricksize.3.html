NAME

ookmaxbricksize - reports the brick size used when initializing a file.  

SYNOPSIS

#include <ook.h>

void ookmaxbricksize(const struct ookfile* of, size_t bsize[3]);

 

DESCRIPTION

ookmaxbricksize() reports the brick size input in the previous ookread(3) or ookcreate(3) call. The bsize argument is filled with the brick size.

Note that any individual brick may be smaller than this brick size. In particular, Ook creates tiny bricks on the edges of data sets to account for the case that the brick size does not evenly divide the domain. However, Ook guarantees that no brick will be larger than the size return from ookmaxbricksize().

 

RETURN VALUE

ookmaxbricksize() technically returns nothing. Instead, it fills the bsize argument.

 

ERRORS

EINVAL
The given of pointer does not point to a valid struct ookfile pointer.

ookread(3), ookcreate(3), ookbricksize(3)
