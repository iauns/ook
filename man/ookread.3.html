NAME

ookread - open a file for reading  

SYNOPSIS

#include <ook.h>

struct ookfile* ookread(struct io interface, const char* filename, 
                        const uint64_t voxels[3], 
                        const size_t bricksize[3], 
                        const enum OOKTYPE type, 
                        const size_t components);

 

DESCRIPTION

ookread gives back a struct ookfile, which is used in basically all other Ook function calls. It allows only read access to a file.

The interface option gives the IO interface to use; see io-interface(7). filename is the filename to read. The number of voxels in the dataset must be given in voxels. bricksize is the desired size of the bricks in the volume. Note this must make sense for the given volume; one cannot have larger bricks than the volume itself. type is the primitive type of the data; Ook uses this and components to figure out how big each element is.

 

RETURN VALUE

On success, ookread() returns a pointer to an opaque structure describing the file. On error, NULL is returned and errno is set appropriately.

 

ERRORS

In general, an exhaustive listing is impossible; ookread() can fail for any of the reasons that interface's open method can fail.
EINVAL
Brick size is larger than volume size.
ENOMEM
No memory available to create opaque structure.
