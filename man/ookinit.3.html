NAME

ookinit − initializes the Ook library

SYNOPSIS

#include <ook.h>

bool ookinit();

DESCRIPTION

ookinit() initializes the Ook library. Every process must call ookinit() before any other Ook function calls.

RETURN VALUE

ookinit() may return false when library initialization fails. The results of Ook operations after an unsuccessful ookinit() call are undefined.

ERRORS

No errors are currently defined.
