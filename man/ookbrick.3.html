NAME

ookbrick - read a brick from an ookfile  

SYNOPSIS

#include <ook.h>

int ookbrick(const struct ookfile* of, size_t bid, void* data);
 

DESCRIPTION

ookbrick() reads a brick from the of ookfile into the memory pointer to by data. The brick to read is described by the bid brick ID. Brick IDs start at 0 and increment until ookbricks()-1, layed out in scanline order. That is, brick ID 0 refers to the brick at coordinate {0,0,0}, brick ID 1 refers to {1,0,0}, and so on, until the maximum number of bricks in X is reached. At that point, the next brick is {0,1,0}, and the count continues in this manner. The 1-dimensional brick ID given to ookbrick() could be calculated from the 3-dimensional brick index by the equation: z*ROWS*COLS+y*COLS+x.

 

RETURN VALUE

ookbrick() returns 0 on success and a nonzero value on error.

 

ERRORS

EINVAL
The given of pointer does not point to a valid struct ookfile pointer, or data is not a valid pointer.
