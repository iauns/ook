NAME

ookclose − flush pending operations and release internal resources for an ook file.

SYNOPSIS

#include <ook.h>

int ookclose(struct ookfile* of);

DESCRIPTION

ookclose() flushes any pending operations for a file opened by ookcreate(), and releases any internal resources allocated from ookcreate() or ookread().

Note that there may be an internal limit to the number of concurrently open files a process may have.

RETURN VALUE

ookclose() returns 0 on success and a non-zero error code on failure.

ERRORS

EINVAL

The given of pointer does not point to a valid struct ookfile pointer.

In addition, ookclose() may fail for any of the reasons that the previously-given io-interface(7)’s close method can fail.
